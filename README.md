# Asteroid Blaster

The initial JavaScript code in this repo was a very rough and ready port of an old Director shockwave game I'd made in 2002. Returning to the repo in 2026 I can see it's time for a complete re-write.

## History

- 2002: The initial code for this game was originally written using Macromedia Director (later Adobe Director) and exported as a Shockwave web game.

- 2017: Adobe Director was offically disontinued. To preserve the game I copied the code I'd written and made a rough translation to JavaScript.

- 2026: Major re-write. Everything in this repo prior to 2026 can be disregarded.
