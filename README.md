# Asteroids

Recreate an Asteroids style game using es6 and css.

The initial JavaScript code in this repo was a straight port from a very old Director shockwave game I'd written around 1999. The quick conversion of the code worked in JavaScript and CSS but it was a very rough and ready port of 20 year old legacy code. Returning to the repo in 2026 I can see it's time for a complete re-write. Instead of making a new repo I'll continue this project with a major re-write.

## History

- 1999: The initial code for this game was originally written using Macromedia Director (later Adobe Director) and exported as a Shockwave web game.

- 2002: The code was used to create an asteroids Shockwave game for the PlayJam website

- 2017: Adobe Director was offically disontinued. To preserve the game I copied the code I'd written and made a rough translation to JavaScript.

- 2026: Major re-write. Everything in this repo prior to 2026 can be disregarded.
